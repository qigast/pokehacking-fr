## Notes
::: tip CONSEIL
La décompilation de cette version est la plus avancée des trois (cinq, en comptant les versions) jeux, que ce soit sur le plan de la documentation, ou les "fonctionnalités". Si vous souhaitez vous lancer dans le source-hacking de la 3G, c'est la version que je vous recommande de choisir.
