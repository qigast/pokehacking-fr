# Pokémon - Version Rouge Feu et Vert Feuille
## Version Française
La décompilation de la version Française, que je maintiens, est disponible sur ce lien:
https://github.com/qigast/pokerougefeu.

::: details DETAILS
Cette décompilation, une fois compilée, produit les ROMs dont les informations suivent:
- [`pokerougefeu.gba`](`https://datomatic.no-intro.org/index.php?page=show_record&s=23&n=1658`)

- [`pokevertfeuille.gba`](`https://datomatic.no-intro.org/index.php?page=show_record&s=23&n=1659`)
- `sha1`: `4b5758c14d0a07b70ef3ef0bd7fa5e7ce6978672`
:::

::: warning ATTENTION
Même si cette version est compatible avec tous les outils (`porymap`, `poryscript`, etc.), elle est *plus ou moins en retard* par rapport à la version Américaine.
:::
